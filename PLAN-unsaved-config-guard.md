# Feature Implementation Plan

**Overall Progress:** `100%`

## TLDR
在保持“启动时自动保存后启动”的前提下，新增行业弹窗未保存拦截，并在运行中保存时给出“下次启动生效”的明确提示。

## Critical Decisions
- Decision 1: 保留启动自动保存 - 不改变现有操作习惯。
- Decision 2: 仅拦截“行业弹窗未保存”后启动 - 聚焦高频误操作场景。
- Decision 3: 运行中保存提示“下次启动生效” - 不引入热更新复杂度。
